## 日本語

「ばりっどーぬ - OpenAPI Valid Dog」（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、以下のポリシーに従って運用されます。

### データの収集

本拡張機能は、ユーザーの個人情報やAPIトラフィックの内容を**収集しません**。

### データの保存

以下のデータはユーザーのローカルブラウザ内（Chrome Storage API）にのみ保存されます：

- 言語設定（日本語/英語）
- 読み込んだOpenAPI仕様書
- フィルタ設定

これらのデータは外部サーバーに送信されることはありません。

### データの共有

本拡張機能は、いかなるデータも第三者と共有しません。

### 権限について

本拡張機能は以下の権限を使用します：

- **storage**: ユーザー設定とOpenAPI仕様書をローカルに保存するため
- **host_permissions (`<all_urls>`)**: DevToolsでネットワークトラフィックを監視するため（監視対象のURLは事前に特定できないため、すべてのURLへのアクセス権限が必要です）

これらの権限は本拡張機能の機能提供のみに使用され、ユーザーデータの収集や外部送信には使用されません。

### お問い合わせ

本プライバシーポリシーに関するご質問は、GitHubリポジトリのIssueにてお問い合わせください。

## English

"ValidDog - OpenAPI Valid Dog" (hereinafter "this extension") respects user privacy and operates according to the following policy.

### Data Collection

This extension **does not collect** any personal information or API traffic content from users.

### Data Storage

The following data is stored only in the user's local browser (Chrome Storage API):

- Language settings (Japanese/English)
- Loaded OpenAPI specifications
- Filter settings

This data is never sent to external servers.

### Data Sharing

This extension does not share any data with third parties.

### Permissions

This extension uses the following permissions:

- **storage**: To save user settings and OpenAPI specifications locally
- **host_permissions (`<all_urls>`)**: To monitor network traffic in DevTools (access to all URLs is required because the URLs to be monitored cannot be predetermined)

These permissions are used solely to provide the extension's functionality and are not used to collect or transmit user data.

### Contact

If you have any questions about this privacy policy, please contact us through the GitHub repository Issues.
